# DebugSettings

这个组件用来简化添加调试开关的流程，方便App快速添加各种调试工具。

目前主要支持类型：

- 开关类(switch)：提供开关状态变化事件处理能力

- 按钮类(button)：提供点击事件处理能力

- 子页面类(subpage): 跳转调试设置子页面能力

## 接入

### SPM方式

```swift
.package(url: "https://github.com/wangzhizhou/DebugSettings.git", branch: "main")
```

### Cocoapods方式

```ruby

pod 'DebugSettings'

```
